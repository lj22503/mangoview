## 二、宏观指标接口 `/v1/market/macro`

### 响应结构

```json
{
  "code": 0,
  "data": {
    "indicators": [/* 指标数组 */],
    "updated_at": "2026-06-26T16:20:00.000000"
  }
}
```

### 指标字段说明

| 字段 | 中文名 | 类型 | 说明 | 示例 |
|------|--------|------|------|------|
| `name` | 指标名称 | string | 显示名称 | `"制造业PMI"` |
| `current` | 当前值 | number\|null | 最新数值 | `50.0` |
| `previous` | 上期值 | number\|null | 上期数值 | `50.3` |
| `direction` | 趋势 | string | `up`/`down`/`flat` | `"down"` |
| `percentile` | 分位 | number\|null | 历史分位（暂无真实值，固定50或null） | `null` |
| `date` | 数据月份 | string | 数据对应月份，格式`YYYY-MM-01` | `"2026-05-01"` |
| `source` | 数据来源 | string | 发布机构 | `"国家统计局"` |
| `available` | 是否可用 | boolean | 数据是否新鲜（≤2个月） | `true` |

### 9 个指标清单

| 序号 | name（API返回） | 中文含义 | 数据源 | 更新频率 | 新鲜度窗口 |
|------|----------------|---------|--------|---------|-----------|
| 1 | `制造业PMI` | 制造业采购经理指数 | 国家统计局 via 东方财富 | 月度（每月最后一天） | 2个月内 |
| 2 | `CPI` | 居民消费价格指数同比 | 国家统计局 via 东方财富 | 月度（每月9日） | 2个月内 |
| 3 | `PPI` | 工业生产者出厂价格指数同比 | 国家统计局 via 东方财富 | 月度（每月9日） | 2个月内 |
| 4 | `GDP增速` | GDP 当期同比增速 | 国家统计局 via 东方财富 | 季度 | 季度窗口 |
| 5 | `M2增速` | 货币供应量M2同比 | 中国人民银行 via 东方财富 | 月度 | 2个月内 |
| 6 | `社融增量` | 社会融资规模增量 | 中国人民银行 via akshare | 月度 | 2个月内 |
| 7 | `社零增速` | 社会消费品零售总额同比 | 国家统计局 via 东方财富 | 月度 | 2个月内 |
| 8 | `出口增速` | 出口金额同比 | 海关总署 via 东方财富 | 月度 | 2个月内 |
| 9 | `固投增速` | 固定资产投资完成额同比 | 国家统计局 via 东方财富 | 月度 | 2个月内 |

### 数据源详情

| 数据源 | 接入方式 | 备注 |
|--------|---------|------|
| 东方财富 `datacenter-web.eastmoney.com` | HTTP API | PMI/CPI/PPI/GDP/M2/社零/出口/固投 |
| akshare `macro_china_shrzgm()` | Python 库 | 社融（东方财富无此指标） |
| DB 优先降级 | SQLite | 有历史数据时优先从 DB 读取 |

### 新鲜度规则

- **月度数据**（PMI/CPI/PPI/M2/社零/出口/固投）：数据月份 ≥ 当前月份 - 2 个月 → 可用
- **季度数据**（GDP）：数据季度 ≥ 当前季度 - 1 季度 → 可用
- 不满足上述条件 → `available: false`，`current: null`，前端显示"数据暂不可用"

### 页面使用情况

| 页面 | 使用的字段 |
|------|----------|
| `/market` 顶部横条 | `name` `current` `direction` `date` |
| `/market` 周期定位卡 | `name` `current` `direction`（筛选 PMI/GDP/PPI） |
| `/market/cycle` 宏观数据表 | 全部字段 |
| `/market/cycle` 趋势验证 | `name` `current` `direction` `source` |
| `/market/cycle` 四周期表格 | `current`（按 name 查找） |
| `/market/industry` | `PMI`（在页面文案中引用当前值） |

## 三、北向资金接口 `/v1/market/north-money`

### 响应结构

```json
{
  "code": 0,
  "data": {
    "date": "2024-08-16",
    "net_buy": -67.7499,
    "buy_amount": 426.4131,
    "sell_amount": 494.163,
    "cumulative_net_buy": 1.7615224900000002,
    "hs300_change": 0.07,
    "available": false,
    "updated_at": "2026-06-26T15:54:00.105537"
  }
}
```

### 字段说明

| 字段 | 中文名 | 类型 | 说明 | 示例 |
|------|--------|------|------|------|
| `date` | 数据日期 | string | 北向资金统计日期 | `"2024-08-16"` |
| `net_buy` | 净买入额 | number\|null | 当日北向净买入（亿元） | `-67.7499` |
| `buy_amount` | 买入成交额 | number\|null | 当日买入总额（亿元） | `426.41` |
| `sell_amount` | 卖出成交额 | number\|null | 当日卖出总额（亿元） | `494.16` |
| `cumulative_net_buy` | 历史累计净买 | number\|null | 历史累计净买额（亿元） | `1.76` |
| `hs300_change` | 沪深300涨跌幅 | number\|null | 当日沪深300涨跌幅（%） | `0.07` |
| `available` | 是否可用 | boolean | 数据是否在7天内 | `false` |
| `updated_at` | 更新时间 | string | 本次API响应时间 | `"2026-06-26T15:54:00..."` |

### 数据源

| 数据源 | 接入方式 | 备注 |
|--------|---------|------|
| akshare `stock_hsgt_hist_em()` | Python 库 | 沪深港通历史日线（数据最新到2024-08-16） |
| DB 优先 | SQLite | 有历史数据时优先从 DB 读取 |

### 新鲜度规则
- 数据日期距今 ≤ 7 天 → `available: true`
- 超过 7 天 → `available: false`，前端显示"北向数据暂不可用"

### 页面使用情况

| 页面 | 使用的字段 |
|------|----------|
| `/market` 信号卡 | `net_buy` `hs300_change` `cumulative_net_buy` |
| `/market/events` 北向资金区 | 全部字段 |

### 当前已知问题
- akshare 北向资金数据最新到 **2024-08-16**（约2年前），已远超7天窗口
- 当前数据不可用，等待新的可用数据源接入

## 四、行业数据接口 `/v1/market/industries`

### 响应结构

```json
{
  "code": 0,
  "data": {
    "industries": [/* 行业数组 */],
    "updated_at": "2026-06-26T16:20:00.000000"
  }
}
```

### 字段说明

| 字段 | 中文名 | 类型 | 说明 | 示例 |
|------|--------|------|------|------|
| `code` | 行业代码 | string | 申万行业分类代码 | `"BK0001"` |
| `name` | 行业名称 | string | 行业中文名 | `"消费"` |
| `cycle_stage` | 周期阶段 | string | 复苏早期/成长期/成熟期/衰退期 | `"复苏早期"` |
| `pe_percentile` | PE分位 | number | 当前PE在历史5年中的分位（0-100） | `28.5` |
| `net_profit_growth` | 净利润增速 | number | 净利润同比增速（%） | `12.3` |
| `penetration` | 渗透率 | number | 产品/服务市场渗透率（0-1） | `0.75` |
| `cr3` | CR3 | number | 前三大公司市场份额（0-1） | `0.42` |
| `weight` | 行业权重 | number | 在整体经济中的占比（0-1） | `0.15` |

### 数据源

| 数据源 | 接入方式 | 备注 |
|--------|---------|------|
| DB | SQLite | 从 DB 读取 `industry_info` / `industry_financials` / `industry_valuation` |
| 降级 | 硬编码 | DB 为空时返回3个示例行业（**生产环境不可用，已禁用**） |

### 页面使用情况

| 页面 | 使用的字段 |
|------|----------|
| `/market` 行业卡 | `name` `net_profit_growth`（top3/bottom3） |
| `/market/industry` 全部 | 全部字段 |
| `/market/industry` 四象限图 | `pe_percentile`（X轴） `net_profit_growth`（Y轴计算） `weight`（气泡大小） |
| `/market/industry` 详情弹窗 | 全部字段 |

### 当前状态
- **DB 当前为空**，无法获取真实行业数据
- 降级硬编码已被禁用 → 前端显示"数据加载失败"
- 需要先完成数据采集才能正常显示
