## 五、events 页面其他数据（前端硬编码）

以下数据为前端 `hardcode`，**非真实 API 数据**：

### 闸门信号 `gateEvents[]`
| 字段 | 说明 |
|------|------|
| `date` | 事件日期（示例） |
| `content` | 事件描述（示例） |
| `industry` | 影响行业 |
| `impact` | 影响方向 positive/negative/neutral |
| `rulerScore` | 三把尺子评分 `{name, score}` |

### 管道信号 `pipeEvents[]`
同闸门信号结构

### 背离信号 `divergeEvents[]`
同闸门信号结构，当前为空数组

### 综合看板（硬编码展示）
| 区块 | 内容 | 性质 |
|------|------|------|
| 周期坐标 | 基钦/朱格拉/库兹涅茨 周期阶段描述 | 硬编码 |
| 估值水位 | PE分位判断 | 硬编码 |
| 边际变化 | 北向资金/PMI等边际描述 | 硬编码 |
