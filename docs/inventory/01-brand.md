## 一、品牌架构

**Mangofolio** = 理财师 AI 工作台（B 端）+ 投资者认知入口（C 端）+ 超级系统站

```
Mangofolio 品牌
├── 个人版（C端）    ← investment-assistant + SaaS 可视化界面
├── 投顾版（B端）    ← mangofolio B端工作台
└── 超级系统站      ← 知识系统 + 通用能力系统 + 数据系统 + 分析引擎
```

**定位对标**：Vanguard（先锋领航）

- **个人版**：个人投资者的"第二大脑"
- **投顾版**：理财师/投顾的专业工作台
- **超级系统站**：底层能力中枢，可被内外调用
