# 13 记忆系统

> 来源：PROJECT_INVENTORY.md 第五章
> 主题：记忆系统设计原则 + 个人版/投顾版独立

---

## 核心原则

**记忆跟着场景和 agent 走，各端独立**

---

## 5.1 个人版记忆系统

**数据**：
- 用户画像（风险偏好、能力圈、投资原则）
- 历史决策（买了什么、为什么买、什么时候卖）
- 想法卡片（关于标的的想法）
- 行为模式（追高、止损过早等）

**特点**：用户私有，隐私优先

---

## 5.2 投顾版记忆系统

**数据**：
- 客户画像（风险偏好、持仓状况、交互历史）
- 理财师工作记录（分析历史、服务记录）
- 话术历史（使用过的话术模板）

**特点**：客户关系数据，需要保密

---

## 5.3 各端记忆独立

```
个人版记忆系统 ← 独立 → 投顾版记忆系统
     ↓ ↓
 用户A的数据   客户B的数据
```

**不同端之间的记忆不互通，各自隔离。**
