## 一、 项目定位与核心逻辑

### 1. 核心定位
**Mangofolio = 基于经典框架的 SaaS 投资辅助工具**

*   **不是**：直接提供买卖建议的投顾平台（合规风险）。
*   **不是**：单纯的数据堆砌（缺乏叙事）。
*   **是**：提供“数据 + 工具 + 框架”的 SaaS 平台，帮助用户建立自己的投资体系。

### 2. 核心逻辑：自上而下
**宏观周期定位 → 行业周期定位 → 寻找机会**

*   **数据层**：提供客观数据（宏观/行业/事件），不做主观判断。
*   **工具层**：将经典投资框架（格雷厄姆/巴菲特/达利欧等）外化为可复用的工具。
*   **配置层**：基于用户画像 + 数据层 + 工具集，通过“黑盒引擎”输出个性化配置方案。

## 三、 页面叙事设计

### 1. 📊 市场看版（数据层）

**叙事焦点**：剥离噪音，建立坐标。解决“信息过载但认知模糊”的冲突。

**页面结构**：

*   **顶部定调**：📅 2026年6月 宏观数据快照（9维度表格：GDP/CPI/PMI等，标注来源与趋势）。
*   **Tab 1：周期定位（我们在哪？）**
    *   **内容**：四周期嵌套数据表（基钦/朱格拉/库兹涅茨/康波）。
    *   **展示**：最新数据 + 历史对照/当前状态。
    *   **原则**：只展示数据，不做主观判断（如“向上/向下”）。
*   **Tab 2：行业大图（行业在哪？）**
    *   **四象限矩阵**：估值分位 vs 净利润增速（气泡大小=权重，颜色=周期阶段）。
    *   **数据表格**：行业 | 周期阶段 | 渗透率 | CR3 | PE分位 | 净利润增速。
    *   **行业秩**：降秩拆解（核心生成器 + 行业性质 + 跟踪要点）。
*   **Tab 3：机会扫描（机会在哪？）**
    *   **边际变化**：闸门/管道/背离事件追踪（三把尺子评分）。
    *   **综合看板**：跨模块联动（周期底部 + 估值低位 + 边际改善）。

**转化钩子**：🔒 历史市场印证 + 深入洞察 → [加入星球解锁]

**页面结构**：

*   **工具分类**：

    *   **行业分析工具**：行业分析仪 (industry-analyst + industry-specialist)。

**转化钩子**：🔒 高级工具（降秩/回测） → [加入星球解锁]

### 4. 📰 扫描报告（服务层）

**叙事焦点**：您负责决策，我们负责盯盘。解决“没时间研究，或盯盘太多被情绪干扰”的冲突。

**页面结构**：

## 四、 注意力工程设计 (Attention Engineering)

### 1. 注意力捕获 (Gap Creation)
*   **首屏 Hook**：“企业家做投资，一半极优秀，一半极拉胯。差别的根源不在认知，在心性。你属于哪一种？”
*   **视觉设计**：左侧“周期定位雷达图”（模糊/锁定），右侧“输入你的投资年限/风险偏好，生成你的投资 DNA”。
*   **原理**：挑战自信，利用损失敏感。

### 2. 注意力锚定 (Attention Anchoring)
*   **微行动**：3 题投资人格测试（情境包裹）。
    1. “如果买入的基金一周跌了 10%，你的第一反应是？”
    2. “你目前投资资金占可投资资产的比例是？”
    3. “你过去 3 年的投资主要收益来源是？”
*   **即时反馈**：生成“投资人格卡片”（如：SRLC 价值守护者），指出盲区（如：“你的盲区是‘处置效应’，当前市场处于复苏早期，可能导致你过早卖出”）。
*   **原理**：插入认知缓冲，记录偏差。

### 3. 注意力留存 (Retention)
*   **资产沉淀**：“你的投资 DNA 已生成。我们将根据你的盲区，在市场出现特定信号时提醒你。”
*   **心理沉没成本**：“你已完成 3 步认知升级，距离‘完整投资框架’还差 2 步。继续完善画像，解锁你的专属配置方案。”
*   **原理**：用户在这里长出了自己的“投资能力”和“偏差记录”，离开等于放弃进化。

### 4. 注意力转化 (Conversion)
*   **价值展示**：在结果页底部，展示“完整配置方案”的轮廓（但关键数据模糊）。
*   **行动号召**：“加入知识星球，获取你的专属配置方案 + 历史市场印证 + 100 次/月 AI 分析额度。”
*   **原理**：用户已经看到了“可能性”，现在需要的是“确定性”和“可执行性”。

## 六、 合规与变现设计

### 1. 合规底线
*   **页面声明**：顶部固定显示“本平台仅提供投资框架与数据参考，不构成任何投资建议。市场有风险，投资需谨慎。请独立判断并自行承担风险。”
*   **去观点化**：所有分析标注“基于框架逻辑推演”，不出现“强烈建议买入/卖出”等主观表述。
*   **黑盒化**：配置中心的具体结果（比例/标的）对免费用户锁定，避免直接输出投资建议。

### 2. 变现路径 (知识星球)
*   **免费用户**：看市场数据、用基础工具、看报告摘要。
*   **星球用户**：
    *   解锁全部高级工具（降秩/回测/事件研判）。
    *   解锁个性化配置方案（比例/标的/分批计划）。
    *   解锁完整报告 + 实时推送。
    *   AI 分析额度（100次/月）。
*   **转化路径**：微信扫码加入星球 → 获取专属账号 → 解锁全部功能。
