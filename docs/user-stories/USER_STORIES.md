# MangoView 用户故事

> **版本**: v1.0.0  
> **日期**: 2026-06-03  
> **状态**: 草案  
> **目标用户**: 3-5 年经验个人投资者  

---

## 一、用户画像

### 核心用户：体系化探索者

| 属性 | 描述 |
|------|------|
| **年龄** | 28-40 岁 |
| **经验** | 3-5 年投资经验 |
| **痛点** | 信息过载但认知模糊，凭感觉买卖，缺乏体系 |
| **需求** | 建立自己的投资体系，有工具辅助决策 |
| **付费意愿** | 中等（愿意为工具付费，但需要看到价值） |
| **技术能力** | 中等（会用 Excel，可能用过同花顺/东方财富） |

---

## 二、用户旅程

### 阶段 1：发现（Awareness）

**用户状态**：知道有问题，但不知道解决方案

```
用户在雪球/知乎看到文章
  ↓
点击链接进入 MangoView
  ↓
看到首屏 Hook："3 年投资，你是在赚钱还是靠运气？差别的根源不在信息，在体系。你属于哪一种？"
  ↓
产生好奇，继续浏览
```

**关键触点**：
- 首屏 Hook 文案："3 年投资，你是在赚钱还是靠运气？差别的根源不在信息，在体系。你属于哪一种？"
- 周期定位雷达图（模糊/锁定）
- "输入你的投资年限/风险偏好，生成你的投资 DNA"

---

### 阶段 2：体验（Experience）

**用户状态**：尝试使用，评估价值

```
完成 3 题投资人格测试
  ↓
生成"投资人格卡片"（如：SRLC 价值守护者）
  ↓
看到盲区提醒："你的盲区是'处置效应'，当前市场处于复苏早期，可能导致你过早卖出"
  ↓
查看免费数据（宏观快照、行业列表）
  ↓
发现高级功能锁定（🔒）
```

**关键触点**：
- 3 题投资人格测试（情境包裹）
- 投资人格卡片（即时反馈）
- 免费数据展示（价值证明）
- 锁定内容（转化钩子）

---

### 阶段 3：转化（Conversion）

**用户状态**：认可价值，考虑付费

```
点击"加入知识星球解锁"
  ↓
扫码加入知识星球
  ↓
获取专属账号
  ↓
解锁全部功能
  ↓
使用配置中心生成专属方案
```

**关键触点**：
- 加入星球按钮（CTA）
- 微信扫码（支付流程）
- 账号激活（即时解锁）
- 配置中心（核心价值）

---

### 阶段 4：留存（Retention）

**用户状态**：已付费，持续使用

```
每日查看市场扫描报告
  ↓
使用工具分析持仓
  ↓
收到信号提醒（基金经理变更/规模变化）
  ↓
参与星球讨论
  ↓
推荐给朋友
```

**关键触点**：
- 每日报告（定时推送）
- 工具使用（持续价值）
- 信号提醒（主动触达）
- 社群互动（归属感）

## 四、关键用户旅程（User Journeys）

### 旅程 1：首次使用 → 加入星球

```
1. 访问首页（首屏 Hook）
2. 完成 3 题测试（投资人格卡片）
3. 查看免费数据（价值证明）
4. 发现锁定内容（转化钩子）
5. 点击"加入星球"（CTA）
6. 扫码支付（微信支付）
7. 账号激活（即时解锁）
8. 使用配置中心（核心价值）
```

**转化漏斗**：
- 访问 → 测试完成：60%
- 测试完成 → 查看数据：80%
- 查看数据 → 点击 CTA：30%
- 点击 CTA → 支付完成：50%
- **总转化率**：7.2%

### 旅程 2：每日使用 → 持续留存

```
1. 打开 MangoView（每日报告推送）
2. 查看市场扫描（宏观数据 + 行业四象限）
3. 使用工具分析（周期定位器/行业分析仪）
4. 查看持仓检查（纪律检查）
5. 参与星球讨论（社群互动）
6. 收到信号提醒（主动触达）
```

**留存指标**：
- 日活（DAU）：目标 30%
- 周活（WAU）：目标 60%
- 月活（MAU）：目标 80%
- 续费率：目标 70%
