## 四、不可用数据及原因

### 1. 指数/个股行情 ❌

**原因**: 阿里云服务器 IP（47.103.44.85）被东方财富限制

**影响**:
- 无法获取沪深 300/中证 500 等指数 K 线
- 无法获取个股实时行情和历史 K 线
- 无法获取行业估值数据（PE/PB/股息率）

**解决方案**:
1. 使用代理服务器（推荐）
2. 使用其他数据源（tushare/聚宽）
3. 手动导入历史数据

### 2. 行业估值数据 ❌

**原因**: 同上，东方财富 API 限制

---

## 五、替代方案

### 方案 A：使用代理（推荐）

```bash
# 配置代理
export HTTP_PROXY=http://your-proxy:port
export HTTPS_PROXY=http://your-proxy:port

# 测试
python test_akshare_data.py
```

**优势**: 解决 IP 限制，获取完整数据  
**劣势**: 需要额外成本

### 方案 B：使用 tushare

```bash
pip install tushare
```

**优势**: 数据质量高，稳定  
**劣势**: 高级数据需付费，有调用限制

### 方案 C：混合方案（推荐）

| 数据类型 | 数据源 | 优先级 |
|---------|--------|--------|
| 宏观数据 | akshare | P0 ✅ 已验证 |
| 北向资金 | akshare | P0 ✅ 已验证 |
| 指数 K 线 | tushare/代理 | P1 |
| 个股行情 | tushare/代理 | P1 |
| 行业估值 | tushare/代理 | P1 |
