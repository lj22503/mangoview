# MangoView 项目规范

> **品牌**: Mangfolio  
> **定位**: 基于经典框架的 SaaS 投资辅助工具  
> **域名**: view.mangofolio.com  
> **MVP 目标**: 2026-06-17

---

## 一、产品概述

### 1.1 核心定位

**Slogan**: 自上而下，看清每一笔投资

自上而下（Top-Down）投资方法论：
1. **周期定位** - 搞清楚宏观周期在哪
2. **行业选择** - 因为周期在这里，这些行业要关注
3. **事件跟踪** - 钱正在往哪跑，哪些信号在发酵

### 1.2 目标用户

个人投资者（3-5 年经验，有体系化需求）

### 1.3 产品边界

| 是 | 不是 |
|----|------|
| 数据 + 工具 + 框架的 SaaS 平台 | 直接提供买卖建议的投顾平台 |
| 帮助用户建立自己的投资体系 | 单纯的数据堆砌 |
| 提供分析方法和决策辅助 | 保证收益的投资产品 |

## 三、页面结构

### 3.1 首页 / (Landing)

- Hero + Slogan
- 三类投资者对比（凭感觉型/消息驱动型/体系化投资者）
- 自上而下三层漏斗（数据层→工具层→配置层）
- 免费功能入口卡片

### 3.2 市场看版 /market

三个独立页面，顶部 Tab 导航切换，叙事串联：

**叙事串联**：
- 周期定位：先搞明白我们在哪
- 行业大图：因为周期在这里，所以这些行业要关注
- 事件跟踪：钱正在往哪跑，哪些信号在发酵

#### 3.2.1 周期定位 /market/cycle

**叙事句**："就像音乐节奏：基钦是快鼓点，朱格拉是中速节拍，库兹涅茨是慢和弦，康波是很长很长的旋律线。先搞清楚现在哪几个周期同时在什么位置。"

**内容**：
- 四周期叠加折线图
- 当前位置卡片（含通俗比喻）

**四周期通俗解释（口语，对孩子说话）**：

| 周期 | 周期 | 类比 | 说明 |
|------|------|------|------|
| 基钦 | 3-4年 | 小卖部进货 | 店里东西多→少→多，三四年一轮 |
| 朱格拉 | 7-11年 | 换蒸笼换机器 | 买设备、换机器，七八年来一次 |
| 库兹涅茨 | 15-20年 | 盖房子 | 盖一轮房子，十五二十年 |
| 康波 | 50-60年 | 大技术革命 | 爷爷有电视，你有AI，一辈子一次大转折 |

**宏观指标通俗解释表**：

| 指标 | 当前值 | 变化 | 分位 | 为啥重要 |
|------|--------|------|------|---------|
| PMI | 50.8 | ↑0.6 | 65% | 制造业热了，50以上=热闹 |
| CPI | 102.3 | ↑0.2 | 58% | 物价小涨了，>100=东西贵 |
| GDP | 5.2% | ↑0.2 | 62% | 经济在长，5%=还行的 |
| PPI | -2.1% | ↓0.3 | 40% | 企业还苦，负数=原材料便宜 |
| 社融 | 3.2 | ↑0.1 | 55% | 借钱的多了，说明人有信心 |
| M2 | 8.7% | ↑0.2 | 72% | 钱变多了，印钞机开动 |

**钩子**：🔒 历史市场印证 + 周期拐点预警 → [加入星球解锁]

#### 3.2.2 行业大图 /market/industry

**叙事句**："基钦周期处于复苏早期，意味着企业正在补库存，这个时候跟'补货'相关的行业会先涨。"

**内容**：
- 行业周期阶段色块（把行业按所处周期阶段放入不同色块）
- 点击色块看该行业本阶段重点指标
- 四象限矩阵（估值 vs 景气度）
- 指标通俗解释

**行业周期色块**：

| 阶段 | 行业 | 本阶段重点看 |
|------|------|------------|
| 复苏早期 | 消费 | 库存变化、居民消费意愿 |
| 成长期 | 科技 | 渗透率提升、研发投入 |
| 成熟期 | 医药 | 防御性、政策免疫 |
| 衰退期 | 金融 | 银行收紧、净息差 |

**四象限矩阵**：高/低估值 × 高/低增速

**指标通俗解释**：
- 📊 估值(PE)：这把刀现在贵不贵
- 📈 景气度：生意好不好（净利润增速）
- 💰 资金流：钱往哪跑（北向/主力净买）
- 🏆 集中度(CR3)：头部公司强不强

**钩子**：🔒 完整行业降秩 + 机构持仓 → [加入星球解锁]

#### 3.2.3 事件跟踪 /market/events

**叙事句**："周期定完了，行业也看明白了。现在看看外面有没有大事要发生——有人在往某个方向调仓，有人在吹风。"

**内容**：
- 资金流（北向资金 + 主力资金）
- 三种信号通俗解释
- 近一周热点事件列表

**三种信号通俗解释**：

| 信号 | 通俗说 | 例子 |
|------|--------|------|
| 🚪 闸门 | 央行/政策大门开了还是关了 | "降准=打开门，钱变多" "加息=关上门，钱变贵" |
| 🛤️ 管道 | 某个方向的水流变了 | "补贴某行业=给它修水管" "限制某行业=堵管子" |
| 📡 背离 | 两个信号打架了 | "油价涨但需求没跟上" "大盘涨但成交量低" |

**钩子**：🔒 完整事件库 + 历史印证 + AI研判 → [加入星球解锁]

## 四、技术架构

### 4.1 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Next.js 14 + Tailwind CSS |
| 后端 | Python FastAPI |
| 数据库 | SQLite |
| 数据源 | akshare（宏观/北向资金/行业板块） |

### 4.2 数据限制（阿里云服务器 IP 被东方财富限制）

| 数据 | 状态 |
|------|------|
| 宏观数据（PMI/CPI/GDP/PPI/M2） | ✅ akshare 可用 |
| 北向资金 | ✅ akshare 可用 |
| 行业板块列表 | ⚠️ 间歇性可用 |
| 指数/个股行情 | ❌ IP 被限制 |
| 行业估值（PE/PB） | ❌ IP 被限制 |

**MVP 策略**：用 akshare 可用数据快速验证，行业估值数据手动导入或后续解决

## 五、合规设计

### 5.1 免责声明

> ⚠️ 本平台仅提供投资框架与数据参考，不构成任何投资建议。市场有风险，投资需谨慎。请独立判断并自行承担风险。

### 5.2 数据来源标注

所有数据标注来源和时间。

### 5.3 黑盒化

配置中心的具体结果（比例/标的）对免费用户锁定，避免直接输出投资建议。
